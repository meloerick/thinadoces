# Regras do Projeto - Sites para Negócios Locais

## Objetivo
Criar sites modernos, rápidos e focados em conversão para pequenos negócios.

## Design
- Estilo moderno e minimalista
- Foco mobile-first
- Visual premium (tipo Apple / Nike)
- Paleta elegante e feminina (quando for estética)

## Estrutura obrigatória
- Hero com CTA forte
- Serviços com cards
- Depoimentos
- Galeria de imagens
- Botão WhatsApp fixo
- Seção de localização

## Código
- HTML semântico
- CSS organizado (BEM ou padrão limpo)
- JavaScript leve
- Performance otimizada

## Conversão
- Sempre incluir botão:
  "Agendar pelo WhatsApp"
- CTAs espalhados no site
- Texto persuasivo

## Extras
- SEO básico
- Animações suaves
- Responsivo perfeito
